
# BMI Calculator

## Description
This project is a BMI (Body Mass Index) calculator developed in Python. It calculates BMI based on user input (age, height, and weight) and classifies the result as underweight, normal weight, overweight, or obese.

## Technologies
- Python
- Streamlit

## Installation
To run this project, you need to install Streamlit:

```bash
pip install streamlit
```

## Usage
Run application with:
```bash
streamlit run IMC.py
